Sistema para solicitar(abrir um chamado) para serviços mecanicos quando se esta viajando.

 Lado cliente-> é feito login(depois do cadastro) entra no sistema e faz chamado enviando sua posição 
 Lado mecanico-> faz login (nao tem cadastro fazer direto no banco) lista todos chamados disponiveis,
	opções
		Ver-> mostra localização do solicitante.
		Antender Chamado-> grava o chamado para o mecanico
		Remover-> remove da lista de chamadas em andamento         
	
Composto por:
	Server-> todos arquivos "RoadSOS-Servidor" e "App" devem estar dentro do servidor de aplicação php
	
	Android-> "My app" arquivos do cordova (trocar numero IP)
